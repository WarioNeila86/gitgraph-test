# Learning how to draw git workflow diagrams using GitGraph.js

- [Website](https://gitgraphjs.com/)
- [Simulated scenarios](https://github.com/nicoespeon/gitgraph.js/tree/master/packages/stories/src/gitgraph-js/)

## Prerequisites
Install node

## Steps to execute
1. Install project dependencies by executing `npm install`
2. Execute `npm start` in order to start web server for graph working on master directly
3. Execute `npm run start-pod` to start web server for graph for pod branches
4. Open [localhost](http://localhost:1234) in your browser
